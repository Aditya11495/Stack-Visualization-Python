## Project Overview

This project demonstrates the working of a **Stack Data Structure** using Python.  
It visually shows how Stack follows the **LIFO (Last In First Out)** principle.

Instead of just printing values in the console, this project uses **Matplotlib** to visualize stack operations step-by-step.

---

## What is a Stack?

A Stack is a linear data structure that follows:

LIFO → Last In First Out

Example:

Push: 10, 20, 30, 40

Stack looks like:

40  ← Top  
30  
20  
10  

If we pop → 40 will be removed first.


## Features

- Stack implementation using Python class
- Push operation visualization
- Pop operation visualization
- Step-by-step graphical updates
- Clean and beginner-friendly code
